# scripts

Helper scripts for demo and development.

- `demo_cold_start.sh` — clean DB, start everything, ingest Tokio
- `chaos.py` — chaos test suite
- `mock_graph_server.py` — WebSocket server pushing fake nodes for dashboard dev (Phase 1 Lane D)
- `fake_webhooks.py` — replay fake GitHub webhook events for demo
- `submit.sh` — generate final submission bundle
