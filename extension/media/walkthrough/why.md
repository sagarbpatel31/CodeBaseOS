# Ask "why is this here?"

Open any source file and either:

- **Hover** a line → click **"Why does this line exist?"**, or
- click the **🧬 Why?** CodeLens above any function / class / definition.

The answer is grounded in the graph and **cites the real GitHub PR, commit, or
issue** — click a citation to jump straight to it. Hovering never spends money;
synthesis runs only when you click.
